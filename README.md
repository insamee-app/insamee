# Components

This lib a set of components design for the app from **insamee**.

## Tech Stack

**Client:** Nuxtjs, Nuxt/Components, TailwindCSS

## Installation

This project uses `yarn`.

```bash
    # install dependencies
    yarn

    # dev server
    yarn dev

    # lint files
    yarn lint
```
